# OKX 项目详细实施表

项目：轻量化加密货币合约回测与半自动信号系统。

交易所纠正：实际交易所固定为 OKX。

## 总目标

做成本地单机版本：
- OKX USDT 永续合约历史数据读取。
- 数据质量审计。
- 1h 入场、4h 判断方向。
- 趋势突破策略。
- 逐仓风控、费用、滑点、资金费估算。
- 训练/验证回测。
- 半自动信号输出。
- 默认不自动实盘下单。
