## 本次目标

继续推进现有14日横截面动量的组合构建研究，但不放宽门槛、不把组合工程包装成新Alpha，也不改变生产信号。新增两项事前冻结实验：排名强度分层权重，以及基于已知OKX资金费的持有成本倾斜。

## 研究一：排名强度分层权重

冻结版本：`MOMENTUM_14D_RANK_CONVICTION_HYSTERESIS6_V1`

规则保持4入6出的成员与方向不变，仅在成员变化时按当前动量排名，为每侧四个成员分配固定权重20%、15%、10%、5%。

结果：

- 基础PF：1.0678；
- 压力成本PF：0.9804；
- 总收益：26.37%；
- 最大回撤：36.75%；
- 平均换手较等权基准增加32.43%；
- 随机时点经验p值：0.0758；
- 参数邻域保持正收益，但历史成本、随机安慰剂和组合增量门禁均失败。

最终决定：`REJECT_AND_ARCHIVE_NO_RESCUE`。不得调整权重梯度、刷新规则或成员规则营救。

## 研究二：资金费持有成本倾斜

冻结版本：`MOMENTUM_14D_FUNDING_CARRY_TILT_HYSTERESIS6_V1`

规则保持4入6出的成员和方向不变，只使用入场前已经结算的最近3次OKX资金费均值，对每侧做轻度持有成本排序。收益前门禁先检查因果覆盖、并列歧义、资金费分散度和目标换手。

收益前结果：

- 资金费得分完整率：100%；
- 成员变化刷新次数：642；
- 并列歧义比例：9.19%，低于15%上限；
- 两侧均有非零资金费分散的比例：97.66%；
- 目标换手较等权基准增加24.68%，超过冻结的15%上限。

最终决定：`REJECT_BEFORE_PNL_NO_RESCUE`。未打开任何未来收益、PF、胜率或回撤，不得修改资金费窗口、并列规则或权重向量营救。

## 研究结论

当前证据进一步说明：

- 给动量排名更强的币增加仓位会显著提高换手并恶化压力成本和回撤；
- 基于已知资金费做温和权重倾斜虽然语义清晰、数据完整，但在当前刷新方式下仍产生不可接受的额外目标换手；
- 现有14日动量原始版与4入6出迟滞版仍是正向研究基准，本次没有新增可进入前向影子的版本。
