## 本次目标

继续研究低换手策略，但不再从已失败权重覆盖层中反复营救。针对现有14日横截面动量，先在不查看收益的条件下比较固定2、3、4、5、7、10、14日成员刷新周期，冻结结构上兼顾换手下降和原信号保留度的3日版本，再打开历史收益和稳健性结果。

该版本不是新独立Alpha，而是现有14日动量的低换手执行变体。它只进入独立研究影子，不修改正式信号、A级判断、飞书正式推送、面板正式区域、杠杆、账户或下单路径。
