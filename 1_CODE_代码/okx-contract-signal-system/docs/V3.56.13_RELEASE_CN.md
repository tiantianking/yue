# OKX 合约信号系统 v3.56.13 发布说明

本版本新增并行前向验收、冻结提前淘汰、失败永久归档和批量候选研究入口，并保持系统只输出信号、不执行交易。

## 主要更新

- 14日动量和固定4入6出继续作为研究级影子运行。
- 新候选通过统一研究门禁后可登记独立前向轨道。
- 冻结14、30、45、60和90日检查点。
- 提前淘汰协议使用固定哈希校验，修改阈值会拒绝运行。
- 严重失败候选自动归档，禁止事后调参营救。
- 新增批量候选工厂 `scripts/run_candidate_factory.py`。
- 新增本地入口 `RUN_PARALLEL_ACCEPTANCE.cmd`。
- 研究级飞书消息明确标记为非A级，并按轨道去重。

## 安全边界

- 不自动下单、不自动持仓、不自动调参、不自动晋级A级。
- 新候选必须先通过既有去重、未来函数、自由度、成本和集中度门禁。
- 正式A级仍需固定前向门槛和人工复核。
