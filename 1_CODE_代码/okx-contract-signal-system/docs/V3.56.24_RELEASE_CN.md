# OKX 合约信号系统 v3.56.24 发布说明

发布日期：2026-06-26

## 本次目标

继续研究现有14日横截面动量的低换手执行改进。此前4入6出已经显著减少成员更替，本次进一步检查：当多空成员完全不变时，是否有必要每天把自然漂移后的持仓重新恢复为等权。

该实验不改变任何信号成员、方向、形成期、持有期或初始权重，也不是独立Alpha，不分配新H编号，不修改正式信号、飞书、面板、杠杆或风控逻辑。

## 冻结版本

`MOMENTUM_14D_MEMBERSHIP_CHANGE_REBALANCE_HYSTERESIS6_V1`

冻结规则：

- 18个具备完整OKX资金费历史的成熟USDT永续；
- 严格闭合15分钟K线重采样为完整4小时K线；
- 14日横截面动量，4入6出迟滞；
- 基准每天在04:00 UTC把4多4空恢复为单币绝对权重0.125；
- 主版本只有在任一币进入、退出或改变多空方向时才整体恢复等权；
- 成员不变时不交易，直接持有自然漂移后的权重；
- 最后按实际漂移持仓计入平仓成本；
- 不设置漂移阈值、定期兜底再平衡、波动触发或单币参数；
- 计入实际OKX资金费、16bp基础往返成本与32bp压力往返成本；
- 执行500次随机时点、方向反转和15分钟延迟检验。

## 历史结果

每日等权基准：

- 基础PF：1.1241；
- 总收益：56.21%；
- 最大回撤：26.93%；
- 平均换手：0.2686；
- 压力成本PF：1.0327；
- 压力成本总收益：6.52%。

仅成员变化时再平衡：

- 基础PF：1.1274；
- 总收益：58.21%；
- 最大回撤：26.47%；
- 平均换手：0.2610；
- 压力成本PF：1.0371；
- 压力成本总收益：8.53%；
- 15分钟延迟PF：1.1225；
- 随机时点经验p值：0.0439；
- 反向PF：0.8088。

## 敞口与交易结构

- 成员路径一致率：100%；
- 日度决策中37.13%无需交易；
- 实际再平衡次数：641次，对比基准1019次；
- 平均绝对净敞口：0.41%；
- 绝对净敞口95分位：2.16%；
- 最大绝对净敞口：7.91%；
- 总敞口5分位：94.97%；
- 总敞口95分位：103.10%。

敞口漂移全部在事前上限内，说明“成员不变时不恢复等权”没有破坏市场中性边界。但换手和交易成本仅降低2.83%，远低于冻结的至少15%改善要求。原因是现有4入6出组合的大部分成本来自真正的成员更替，而不是成员不变时的日度等权微调。

## 最终决定

`REJECT_AND_ARCHIVE_NO_RESCUE`

历史收益、压力成本、三个时间段、净敞口、总敞口、随机时点、方向反转和15分钟延迟均通过，但结构与增量价值门禁失败。基础PF、压力PF、收益和回撤虽略有改善，幅度不足以构成冻结的实质执行优势。

禁止增加漂移阈值、每N日兜底再平衡、只重平衡部分成员、波动触发、币种例外或降低15%成本改善门槛来营救本版本。

失败证据已写入桌面 `失败策略/MOMENTUM_14D_MEMBERSHIP_CHANGE_REBALANCE_HYSTERESIS6_V1`，并登记到研究家族库。
