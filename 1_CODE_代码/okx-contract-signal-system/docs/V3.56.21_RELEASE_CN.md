# OKX 合约信号系统 v3.56.21 发布说明

发布日期：2026-06-26

## 本次目标

在不虚构新Alpha、不降低门槛的前提下，继续研究现有14日横截面动量的两种组合构建改进，并把全部冻结协议、随机检验、资金费、失败证据和去重指纹固化到项目中。

## 新增研究一：板块限额组合

冻结版本：`MOMENTUM_14D_SECTOR_CAP2_HYSTERESIS6_V1`

- 18个具备完整OKX资金费历史的成熟永续；
- 保持14日动量与4入6出迟滞；
- 每侧同一固定板块最多2个币；
- 计入实际OKX资金费、16bp基础往返成本与32bp压力往返成本；
- 执行500次随机时点、方向反转、15分钟延迟、三个参数邻域和组合增量检验。

结果：

- 基础PF：1.1412；
- 压力PF：1.0354；
- 总收益：62.70%；
- 最大回撤：24.23%，优于等权基准的26.93%；
- 随机时点经验p值：0.0379；
- 其余历史、稳健性、邻域和组合增量门禁均通过；
- 平均换手较基准增加17.57%，超过事前冻结的15%上限。

最终决定：`REJECT_AND_ARCHIVE_NO_RESCUE`。不得通过放宽换手上限、改变板块划分、修改上限或持有期进行营救。

## 新增研究二：7日左尾风险加权

冻结版本：`MOMENTUM_14D_ES7_RISK_WEIGHT_HYSTERESIS6_V1`

- 选币和方向完全继承14日动量4入6出；
- 只在成分变化时按过去7日最差5%小时收益的期望损失做逆风险分配；
- 单币绝对权重上限20%；
- 同样计入实际资金费、基础与压力成本，并执行完整随机、反向、延迟、邻域和组合增量检验。

结果：

- 基础PF：1.1075；
- 压力PF：1.0121；
- 总收益：45.91%；
- 最大回撤：27.61%；
- 结构、成本、分段、延迟和参数邻域通过；
- 随机时点经验p值：0.0559，高于冻结的0.05上限；
- 与现有等权影子50/50组合后，最大连亏由10增至13，且没有任何冻结的增量改善项通过。

最终决定：`REJECT_AND_ARCHIVE_NO_RESCUE`。不得选择历史更优窗口、修改尾部分位、权重上限或刷新规则营救。
