# v3.56.7 本地影子组合接入说明

`D:\JIAOYI-CX\1_CODE_代码\okx-contract-signal-system`

影子组合已经直接接入桌面 GUI 的扫描循环，不需要单独启动第二个 Python 进程。

## 冻结研究规则

- 数据源：闭合 15m K 线。
- 4h 聚合：必须由同一 4h 区间内 16 根连续、唯一、闭合的 15m K 线组成。
- 成员一：4h Donchian 24，EMA10/EMA50 方向，7 日横截面强弱，成交量比过滤。
- 成员二：VCB_A 波动压缩后扩张突破。
- 横截面参考集固定为 BTC、ETH、SOL、XRP、BCH、LINK、LTC、UNI、ARB、ETC、APT、ATOM。
- 任一固定参考币种缺失或 4h 时间不一致时，影子通道暂停；正式主链继续运行。
- 参数、参考币种和 blind 声明由启动校验锁定，修改后影子模块拒绝启动。

## 隔离边界

影子组合只写入：

- `outputs/shadow_ensemble_status.json`
- `outputs/shadow_ensemble.sqlite3`

它不导入、不写入：

- approved runtime manifest
- 正式 signal lifecycle SQLite
- 正式 notification outbox
- 飞书正式通知
- 账户和下单模块

## Dashboard

Dashboard 增加独立的“4h影子组合”面板，显示：

- 当前状态
- 最新完整 4h 时间
- 横截面可用币种数
- 本轮新观察数
- 待入场、活动和已闭合数量
- 估算净 R 与 PF

## 第一次启动

影子数据库为空时会静默回放最近 42 根完整 4h K 线，用于恢复可能仍在观察期内的状态。预热记录标记为 warmup，不进入前向统计，也不会产生正式通知。

## 启动与检查

运行：

```powershell
cd D:\JIAOYI-CX\1_CODE_代码\okx-contract-signal-system
.\start.bat
```

检查：

```powershell
Get-Content .\outputs\shadow_ensemble_status.json -Encoding UTF8
```

正常状态应为 `running`。固定参考集缺失时会显示 `incomplete_frozen_reference_universe`；时间错位时显示 `frozen_reference_bar_misaligned`。
