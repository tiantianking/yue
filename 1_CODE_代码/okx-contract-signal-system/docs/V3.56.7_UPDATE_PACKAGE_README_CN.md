## 主要更新

- Dashboard 将 Python 实际推送许可 `push_allowed` 与运行健康 `operational_ready` 分离。
- Dashboard stale symbols 判定合并 startup、runtime、closed-backfill 证据，并要求 closed-backfill 足够新鲜才可作为权威替代。
- `RealtimeDataStore.append_candle()` 增加顺序追加快速路径和同时间戳原位替换语义，已闭合 K 线不会被后续 open 更新降级。
- 版本统一提升到 `3.56.7`，approved strategy version 同步提升到 `3.56.7`。
- `SOURCES.txt` 恢复完整覆盖 Python 模块、测试和文档，并新增发布完整性回归测试。
