# OKX 合约信号系统 v3.56.23 发布说明

发布日期：2026-06-26

## 本次目标

继续研究现有14日横截面动量的低换手执行改进。根据此前收益盲审计中固定的30日成交额前12名规则，本次只把流动性用于新成员准入，不允许流动性变化强制退出已有成员；原有14日动量、4入6出、方向、持仓数量和等权权重全部保持不变。

该实验不是独立Alpha，不分配新H编号，不修改正式信号、飞书、面板、杠杆或风控逻辑。

## 冻结版本

`MOMENTUM_14D_LIQUIDITY_ADMISSION12_HYSTERESIS6_V1`

冻结规则：

- 18个具备完整OKX资金费历史的成熟USDT永续；
- 严格闭合15分钟K线重采样为完整4小时K线；
- 14日横截面动量，4入6出迟滞；
- 过去180根4小时K线、约30日的OKX quote volume均值，整体延迟一根4小时K线；
- 只有流动性排名前12的币可以成为新成员；
- 已有成员仍只按动量退出规则处理，流动性不得强制卖出；
- 中心参数固定为12，只审计10和14两个邻域，不择优；
- 计入实际OKX资金费、16bp基础往返成本与32bp压力往返成本；
- 执行500次随机时点、方向反转、15分钟延迟、邻域和50/50组合增量检验。

## 历史结果

基准4入6出：

- 基础PF：1.1241；
- 总收益：56.21%；
- 最大回撤：26.93%；
- 平均换手：0.2686。

流动性入场版本：

- 基础PF：1.1284；
- 总收益：53.20%；
- 最大回撤：25.17%；
- 平均换手：0.3486；
- 压力成本PF：1.0127；
- 压力成本总收益：-1.09%；
- 随机时点经验p值：0.0279；
- 15分钟延迟PF：1.1272；
- 10和14两个固定邻域均为正，邻域PF中位数1.1456。

覆盖层改变组合的时点占比为96.57%，但平均换手相对基准增加29.77%，与“至少降低5%换手”的事前目标方向完全相反。

## 最终决定

`REJECT_AND_ARCHIVE_NO_RESCUE`

历史成本、随机时点、方向反转、延迟、参数邻域和组合增量门禁均通过，但结构门禁失败。PF小幅提高和回撤小幅改善不足以覆盖错误的换手方向；不得放宽换手目标，也不得修改30日窗口、前12名门槛、改成强制流动性退出、挑选10或14的历史较优结果、删除币种或增加价格/波动过滤器营救。

失败证据已写入桌面 `失败策略/MOMENTUM_14D_LIQUIDITY_ADMISSION12_HYSTERESIS6_V1`，并登记到研究家族库。
