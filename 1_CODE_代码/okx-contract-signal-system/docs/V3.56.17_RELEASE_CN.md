# OKX 合约信号系统 v3.56.17 发布说明

发布日期：2026-06-26

## 本次修复

### 当前观察面板不再显示已终止历史记录

v3.56.16 已正确把旧 HYPE 信号更新为 `STOP_REACHED`，但桌面 GUI 仍会把所有生命周期记录，包括已终止历史记录，继续显示在“信号观察记录”主面板中。

v3.56.17 将该面板改为“当前有效信号观察”，只显示：

- `TRIGGERED` 且尚未终止的记录；
- `CONFIRMED` 且尚未终止的记录。

以下终止结果不再显示在当前观察面板：

- `STOP_REACHED`
- `TARGET_REACHED`
- `TIMEOUT_RESULT`
- `CENSORED`

历史记录仍完整保存在 SQLite 生命周期数据库和审计事件链中，不删除、不覆盖，也不修改历史证据。
