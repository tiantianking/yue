# OKX 合约信号系统 v3.56.16 发布说明

发布日期：2026-06-26

## 本次修复

### 1. 修复已确认信号无法结束的问题

旧生命周期状态机只会让未确认信号失效或过期。已进入 `CONFIRMED`、且没有止盈字段的记录，即使后续闭合K线触及止损或超过最大持有期，也会长期停留在活跃状态。

v3.56.16 现在会：

- 对已确认且无止盈字段的信号，使用后续闭合K线高低价检查分析止损；
- 触及止损时记录 `STOP_REACHED`；
- 超过固定最大持有期时记录 `TIMEOUT_RESULT`；
- 只使用闭合K线，不读取未闭合K线。

### 2. 启动时先用本地缓存修复生命周期

桌面程序加载本地历史数据后，会先重放仍处于活跃状态的信号生命周期，再执行在线补线。即使网络补线失败，已有本地闭合K线仍可终止过期或止损记录，避免旧信号长期冒充当前信号。

### 3. 阻止同币同向重复活跃信号

实时扫描入口现在会检查同一合约、同一方向是否已有未终止信号。若存在，新的重复候选不会进入推送队列，也不会新增第二条活跃观察记录。

底层历史账本仍允许保存多个独立历史记录，去重只作用于实时扫描入口，避免影响研究与审计用途。

### 4. 改善桌面展示

“信号观察记录”最后一列由“信号周期”改为“信号时间”，并显示北京时间。历史入场价与当前收盘价不再缺少时间背景。

## 验证

- 生命周期与扫描服务核心回归测试全部通过；
- 覆盖无止盈信号止损、无止盈信号超时、同币同向活跃去重、缺失最新K线时仍更新生命周期、未来闭合K线拒绝等边界；
- 未加入订单、持仓、账户或自动交易功能。
